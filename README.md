# F1 Analyzer V2

## Project description
This project groups together different 
types of data analysis graphs for the official F1® sessions
(free practices, qualifying, races, pre-season tests).

If you install it, you will be able to save the graphs,
and analyze them yourself to make your predictions, and
have more insights about F1® races.

I'd like to note that the graphs were 100% inspired by 
[FDataAnalysis](https://www.instagram.com/fdataanalysis/).
The code was written by me, but it just reproduces his work
and therefore the graphs might not look exactly the same.
